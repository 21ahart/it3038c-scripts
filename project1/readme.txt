calculator.py is a program built to compute two intager numbers

How use:
-Find the file path of this file and execute it within powershell by entering "py (file path)"
-You must format your expression with spaces between the operator and the two numbers
Example: "10 + -9"
-You must use integer values for this calcualtor. Double and float values will throw
errors.
-The program accepts any programming operator " +, -, *, /, %"
-You can keep on computing numbers until you type in "exit" at which point the program
will close
